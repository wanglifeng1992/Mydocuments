第21章  实战桌面搜索系统

本章以一个桌面搜索系统为例，介绍单机模式的程序开发方法，重点是如何利用.NET 控件来实现对磁盘文件系统的访问以及对文件进行管理。

重点从需求分析、运行设计到代码实现过程分别描述。通过本节学习，可以加深对技术的理解，同时掌握技术的实际运用。

其中的运行设计中详细描述了应用系统的功能要求，在阅读本章过程中，读者可以在阅读了本节之后，尝试编写程序实现这些功能，然后再对照后面的代码编写部分，通过比较研究，更能充分理解应用系统开发的技术和思想。

21.1  需求分析

经常遇到对计算机上文件的访问，通常使用系统的资源管理器，但是有的时候需要对长期不使用的文件加以处理，以便节省磁盘空间。本软件的功能要求就是实现对文件进行整理的功能，以便补充系统资源管理器缺少的功能。这些功能包括：

² 选择磁盘目录。

² 查找指定日期范围（一个月）内的没有被修改过的文件，也就是很久没有使用的文件。

² 对这些文件进行删除。

² 列出所有符合条件的文件清单。

下面的代码实现了对树节点信息的构建，实际上是把自动获取的磁盘驱动器的标识号显示出来，形成类似资源管理器的功能。

          void SolveTreeNodes (TreeNode root)

                   {

                   int i;

            try

            {

                string[] d = Directory.GetDirectories(root.Tag.ToString());

                // 创建多个节点...

                TreeNode[] tn = new TreeNode[d.Length];

                // 填充逻辑驱动器信息...

                for (i = 0; i < d.Length; i++)

                {

                    // 添加节点标题...

                    tn[i] = new TreeNode();

                    tn[i].Text = this.getRelativePath(d[i]);

                    tn[i].Tag = d[i];

                    tn[i].ImageIndex = 1;

                }

                // 完成节点信息添加

                root.Nodes.AddRange(tn);

            }

            catch { }

          }

          string getRelativePath (string x)

          {

                   int i;

                   for (i=x.Length-1 ; i>=0 ; i--)

                   {

                            if (x[i] == '\\')

                            {

                                      ////[雨林木风2] System.Console.WriteLine("String found" + i);

                                      break;                           

                            }

                   }

                   // 返回已经过滤的字符串

                   return x.Substring(i+1);

          }

Windows 窗体 TreeView 控件以类似于在 Windows 资源管理器左窗格中显示文件和文件夹的方式显示节点的层次结构。每个节点可能包含称为子节点的其他节点。父节点或包含子节点的节点可以展开或折叠的方式显示。如果将树视图的 CheckBoxes 属性设置为 true，则在显示树视图时，节点旁边还可带有复选框。然后，通过将节点的 Checked 属性设置为 true 或 false，就可以编程方式选定或清除节点。

TreeView 控件的主要属性是 Nodes 和 SelectedNode。Nodes 属性包含树视图中的顶级节点列表。SelectedNode 属性设置当前选定节点。节点旁边可显示图标；从树视图的 ImageList 属性中命名的 ImageList 控件获取图像。ImageIndex 属性为树视图中的节点设置默认图像。

要通过程序确定单击了哪个 TreeView 节点，可以使用 EventArgs 对象返回对已单击的节点对象的引用。通过检查 TreeViewEventArgs 类（它包含与事件有关的数据），确定单击了哪个节点。例如：

21.6  多线程文件检索模块（MultithreadedFileInfo.cs）

本模块实现了利用多线程进行文件检索处理的功能。由于文件检索是复杂的处理过程，涉及到多目录、多文件递归处理，采用线程技术可以防止程序进入频繁的过程调用，导致类似死机的状态。

          {

                   {

            {

          int getMonthsTillNow ( DateTime d )

          {

                   try

                   {

                            long ticks = DateTime.Now.Ticks - d.Ticks;

                            DateTime dt = new DateTime(ticks);

                            return (dt.Month + (dt.Year-1)*12 -1);

                   }

                   catch(Exception e)

                   {

                   e.ToString();       

                   }

                   // 返回错误信息

                   return -1;

                   }

          string[] FormatFileItem (System.IO.FileInfo fi)

          {

                   string[] returnString = new string[6];

                   returnString[0] = fi.Name;

                   // 检查文件多久没有修改过                

                   int Write = this.getMonthsTillNow(fi.LastWriteTime);

                   if (Write == -1)

                   {

                            returnString[4] = "没有修改";

                   }

                   else

                            returnString[4] = "" + Write + " 个月";

          returnString[5] = fi.DirectoryName;

                   int Access = this.getMonthsTillNow(fi.LastAccessTime);

                   if (Access == -1)

                   {

                            returnString[1] = "没有访问";

                   }

                   else

                            returnString[1] = "" + Access + " 个月";

                   returnString[3] = fi.Extension;

                   // 计算文件大小

                   if (this.CallingProgram.SizeEnabled)

                   {

                            // 获取文件长度

                            double sz = fi.Length;

                            sz/=(1024*1024);                                    

                            returnString[2] =  System.Math.Round(sz,2) + "Mb";

                   }

                   else

                            returnString[5] = "不计算";
